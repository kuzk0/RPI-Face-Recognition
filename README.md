# Система распознавания лиц в реальном времени на Raspberry Pi

Этот проект реализует систему распознавания лиц в реальном времени с использованием Raspberry Pi, камеры Raspberry Pi и модели ONNX-based InsightFace. Он захватывает изображение с камеры, сравнивает его с предварительно обученной базой лиц и определяет, кто на фото.

## Что делает проект

- Захватывает изображение с помощью модуля камеры Raspberry Pi.
- Извлекает эмбеддинги с помощью ONNX-модели InsightFace (например, `w600k_r50.onnx`).
- Сравнивает полученные векторы с заранее сохранёнными эмбеддингами из папки `Dataset/`.
- Отображает результат на экране: прямоугольник вокруг лица с именем и уровнем уверенности.
- Поддерживает несколько известных пользователей, организованных по структуре папок.

## Установка и настройка

### Требования
- Raspberry Pi (желательно 3B+, 4 или 5)
- Модуль камеры Raspberry Pi v2 (поддерживает камеру v1, но качество ниже)
- ОС: Raspberry Pi OS Bullseye/Bookworm (Lite или Desktop)

### Шаги установки

1. **Обновите систему**
   ```bash
   sudo apt update && sudo apt upgrade -y
   ```

2. **Установите зависимости**
   ```bash
   sudo apt install python3 python3-pip libatlas-base-dev libhdf5-dev libjpeg-dev
   pip3 install -r requirements_rpi.txt
   ```

3. **Активируйте камеру**
   ```bash
   sudo raspi-config
   # Interface Options → Camera → Enable
   # reboot
   ```

4. **Загрузите ONNX-модель**
   Скачайте одну из рекомендуемых моделей InsightFace (например, `buffalo_m/w600k_r50.onnx`) по [ссылке](https://github.com/deepinsight/insightface/releases/tag/v0.7). Поместите в папку `./models/buffalo_m/`.

   Рекомендуемые модели:
   - ✅ `w600k_r50.onnx` — лучший баланс точности и производительности.
   - ⚠️ `w600k_mbf.onnx` (buffalo_s или buffalo_sc) — чуть быстрее, но менее точно.
   - ❌ `glintr100.onnx`, `buffalo_l` — слишком тяжёлые для Raspberry Pi.

5. **Подготовьте датасет**
   См. раздел **"Особенности работы с датасетом"** ниже.

6. **Запуск**
   ```bash
   python3 recognizer_pi.py
   ```

## Особенности работы с датасетом

В папке `Dataset/` должны находиться поддиректории, каждая из которых отвечает за одного человека.
- Название поддиректории используется как имя распознаваемого человека (например, `andrey`, `max`).
- Внутри каждой такой директории — фото лица этого человека в формате `.jpg` или `.png`. Имена файлов могут быть любыми.
- Чем больше качественных фото одного человека вы добавите — тем стабильнее и надёжнее будет распознавание (система усредняет эмбеддинги при сравнении).
